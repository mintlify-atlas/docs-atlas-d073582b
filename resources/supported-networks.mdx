## Programmatic Access

You can access network information programmatically using the SDK constants:

```typescript
import { 
  SUPPORTED_CHAINS,
  MAINNET_CHAIN_IDS,
  TESTNET_CHAIN_IDS,
  CHAIN_METADATA 
} from '@avail-project/nexus-core';

// Access specific chain IDs
console.log(MAINNET_CHAIN_IDS.ETHEREUM); // 1
console.log(MAINNET_CHAIN_IDS.BASE); // 8453

// Get chain metadata
const ethMetadata = CHAIN_METADATA[1];
console.log(ethMetadata.name); // "Ethereum"
console.log(ethMetadata.nativeCurrency); // { name: "Ether", symbol: "ETH", decimals: 18 }
```

<Info>
  All networks support native token transfers and bridging. Swap functionality is only available on select networks as indicated in the tables above.
</Info>
